Пуляев Евгений 2 курс 4 группа

Название проекта:  клиент-серверное приложение, реализующее функционал тестирования по учебной дисциплине.

Цель:  разработка клиент-серверного приложения для тестирования по учебной дисциплине. Целью разработки является упрощение процесса создания тестов для обучающихся, а также нтестирования самих обучающихся.

Ожидаемые результаты:  разработанное и размещенное на веб-хостинге Sprinthost клиент-серверное веб-приложение, удобное и быстрое содание и доступ к тестам по различным учебным дисциплинам

Задачи:  для достижения цели работы должны быть решены следующие задачи:

  1. осуществить постановку задачи, выделить требования к приложению;
  2. изучить современные средства разработки клиент-серверных приложений;
  3. определить требования и спроектировать клиент-серверное приложение;
  4. реализовать и протестировать клиент-серверное приложение.

Методы решения задач:  обзор и изучение возможных решений и способов реализации при помощи ресурсов сети интернет, использование специальных инструментальных средств разработки клиент-серверного приложения.

Инструментальные средства:  язык веб-разработки php, система управления базами данных MySQL, бесплатный хостинг Sprinthost.

Функционал:  клиент-серверное приложение будет представлять из себя сайт, на котором, в зависимости от роли пользователя, которую он выбирает при регистрации, будет размещен либо список всех тестов и их статус по доступным тестируемому дисциплинам (в случае роли обучающийся), либо редактор тестов с возможностью создания, редактирования и удаления тестов и прав доступа к ним ( в случае роли преподаватель). Для роли администратор будет предоставлена возможность создания дисциплин и предоставления доступа к ней одному или нескольким преподавателям.
